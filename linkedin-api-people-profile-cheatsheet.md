# LinkedIn API vs NinjaPear Person Profile Cheat Sheet

## Official LinkedIn API
- 3-legged OAuth for member-authorized access
- 2-legged access requires approval / partner path
- Limited profile fields for most developers

## NinjaPear Employee API
- Input by work email
- Input by name + company
- Input by role + company
- Returns profile data from public web sources without scraping LinkedIn
- Related primitives: work email finder, similar people, company profile
